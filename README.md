# Servers

This repos is to have most of config files needed to rebuid and make servers for Peer-Network.
Having these files allows us to validate and create server to match the the 3 main evironments.

There is three evironments needed by Peer.
* Prod (Production)
* Beta (Testing)
* Dev (Developement)
